# Codex Skills

Public Codex skills shared from this repository.

## notion-writer

`notion-writer` helps Codex turn notes, drafts, and product thinking into polished Notion documents. It includes a reusable Notion API template for replacing or appending structured page blocks.

Install by copying the `notion-writer/` folder into your Codex skills directory:

```text
~/.codex/skills/notion-writer
```

Then invoke it with:

```text
Use $notion-writer to update a Notion product document from my notes.
```

Secrets are not included. Set `NOTION_TOKEN` or `NOTION_TOKEN_FILE` locally when adapting the API template.
